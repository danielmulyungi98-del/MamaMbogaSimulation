# Mama Mboga Simulation System

## Description
This project is a Mama Mboga Simulation System developed using **VB.NET** and **Windows Forms**
in **Microsoft Visual Studio 2022** as part of an academic assignment.

The application simulates basic interactions in a small vegetable business (Mama Mboga),
such as customer arrival, price negotiation, payment, and stock updates through a graphical user interface.

- Windows Forms
- .NET Framework

## IMPORTANT: How to Open the Project (Please Read)
NB **Do NOT open the project by selecting the folder.**

To correctly view the Forms, labels, buttons, and designer view:

1. Download the project from GitHub (Code → Download ZIP) or clone the repository.
2. Extract the ZIP file.
3. Open **Microsoft Visual Studio 2022**.
4. Click **Open a project or solution**.
5. Select and open the file:
6. 
6. Once opened, in **Solution Explorer**, right-click `Form1.vb` and select **View Designer**.
7. Press **F5** or click **Start** to run the application.
