# Mama Mboga Simulation System

## Description
This project is a Mama Mboga Simulation System developed using VB.NET as part of an academic assignment.
The system simulates the basic operations of a small vegetable vendor (Mama Mboga) using a
Windows Forms application.

## Technologies Used
- VB.NET
- Windows Forms
- .NET Framework
- Microsoft Visual Studio 2022

## Requirements
To run this project, you need:
- Windows Operating System
- Microsoft Visual Studio 2022
- .NET Framework

## How to Run the Project
1. Download the project from GitHub or clone the repository.
2. Extract the files if downloaded as a ZIP.
3. Open the `MamaMbogaSimulation.sln` file using Microsoft Visual Studio 2022.
4. Press **F5** or click **Start** to run the application.

## Project Structure
- `MamaMbogaSimulation/` – Main source code files
- `MamaMbogaSimulation.sln` – Visual Studio solution file
## Features
- Simple graphical user interface
- Simulation of Mama Mboga daily operations
- User interaction through Windows Forms
